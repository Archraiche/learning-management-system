1. Download XAMPP Here: https://www.apachefriends.org/
2. Install XAMPP
3. Open XAMPP then click start on Apache and MySQL
4. Open browser and type: localhost/phpmyadmin/
5. Create a New database named "capstone" 
6. After creating, click Import and choose the file "capstone" sa database folder. 
7. Click Import. 


Locating files:

1. Extract the zip file to D:\Program Files\xampp\htdocs   (basta dapat nasa htdocs yung folder na "capstone")


Opening:
1. type localhost/capstone/login.php (you will be prompted sa login page)
